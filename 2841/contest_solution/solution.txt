1. найти максимальную сумму, если таковую возможно найти, при условии m - количество массивов, k - количество уникальных элементов
2. в качестве решения - используем sliding window
3. считаем сумму от 0 до k, это текущее окно, это же окно сохраняем в виде очереди, чтобы не пересчитывать сумму каждый раз, так же в
4. unordered map сохраняем окно - размер мапы - количество уникальных элементов.
5. далее просто двигаемся по массиву последний элемент в очереди удаляем, его же удаляем из мапы и суммы
6. новый элемент добавляем - получаем новую сумму, новое количество элементов.
7. если количество уникальных элементов больше m, то сумму добавляем в мультисет, последний элемент мультисета будет ответом.
